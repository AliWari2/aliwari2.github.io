Projets - Ali Wari

Projets Réalisés

  • Site de location immobilière (2024) : Développement d’un site permettant aux propriétaires de publier des annonces de logement, avec des fonctionnalités pour les chercheurs de logements, telles que la consultation des annonces, l’ajout aux favoris et la possibilité de contacter directement les propriétaires.
    Langages utilisés : React.js, Node.js, Express.js.
  • Intégration de maquettes de site internet (2023) : Conversion de designs graphiques en pages web interactives, en assurant une expérience utilisateur fluide et ergonomique. Ce projet visait à respecter les normes d’accessibilité et de performance web.
    Langages utilisés : HTML, CSS, JavaScript.
  • Application web de validation de compétences (2024) : Création d’une application permettant aux responsables de formation d’évaluer les participants à travers un jeu de questions-réponses progressif. Chaque bonne réponse permet d’avancer dans le parcours jusqu’à la fin du test.
    Technologies utilisées : HTML, CSS, Bootstrap, JavaScript, PHP.
    Code source : GitHub.
  • Application de gestion de formation (2024) : Projet universitaire en groupe visant à créer un système de gestion de formations. Les étudiants peuvent s’inscrire à une année de formation, choisir leurs unités d’enseignement et consulter les messages. Les responsables peuvent gérer les étudiants, les unités et organiser les groupes automatiquement.
    Technologies utilisées : Java, Git, JUnit, JavaFX.
  • Application de floutage d’image (2023) : Développement d’un logiciel permettant de flouter des images BMP. L’application permet de sélectionner une image depuis l’explorateur de fichiers et d’appliquer différents niveaux de flou en fonction des besoins.
    Technologies utilisées : Programmation en C, Glade.
    Code source : GitHub.
  • Création du jeu Space Invaders (2024) : Développement d’une version du célèbre jeu Space Invaders en utilisant la programmation orientée objet. Les fonctionnalités incluent le déplacement des aliens, la logique de tir, la détection des collisions et la gestion des scores. Le code a été structuré en classes pour une meilleure maintenabilité.
    Technologies utilisées : Python, Tkinter.
